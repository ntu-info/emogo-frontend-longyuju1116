# Daily Mood & Video Diary App

This repository contains the source code for the Expo Router MWE assignment. This app helps users track their daily life by rating their mood and recording a 1-second video snapshot every day.

## üì± Project Overview
The app includes the following key features:
* **Daily Push Notifications:** Automatically reminds the user to log their daily entry.
* **Mood Tracking:** Allows users to rate their current mood score.
* **1-Second Video:** Records and saves a 1-second video clip to capture the moment.
* **Data Export:** 
    * Exports mood records as **CSV** files.
    * Exports recorded clips as **MP4** video files.
